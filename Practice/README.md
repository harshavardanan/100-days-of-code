# Practice

Scratch space for working through problems, trying new ideas, and experimenting with code before moving solutions to their category folders.

## What goes here
- First attempts at a problem before the solution is clean
- Experimental code and language exploration
- Multi-language practice (Python, Java, etc.)

## Workflow
1. Solve the problem here first
2. Once the solution is clean and understood, move it to the appropriate category folder
3. Keep this folder uncluttered — delete or archive old scratch files
